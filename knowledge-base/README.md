# Knowledge Base

This folder is the single source of truth about **you** — your background, career,
and projects. Dump everything here (raw notes, bullet points, pasted text, whatever
form is easiest), and I'll use it to populate the site's pages:

- `about.md` → the **About & Journey** page
- `projects.md` → the **Projects** page
- `contact.md` → the **Get in Touch** page
- everything here → the context the **Ask AI** chat answers from

## How to use it

1. Open any file below and fill it in. Don't worry about formatting, grammar, or
   completeness — bullet points and rough notes are fine. Paste your resume, LinkedIn
   text, project descriptions, anything.
2. When you're ready, tell me "the knowledge base is updated" (or point me at
   specific files) and I'll turn the raw material into polished page content.
3. You can keep adding over time. Re-run the same request and I'll refresh the pages.

## Files

| File | Feeds | What goes in it |
|------|-------|-----------------|
| `profile.md` | About, Chat | Who you are, one-liner, location, current role, interests |
| `about.md` | About, Chat | Career story, background, values, what you're good at |
| `experience.md` | About, Projects, Chat | Jobs, roles, dates, responsibilities, achievements |
| `projects.md` | Projects, Chat | Individual projects: what, why, tech, impact, links |
| `skills.md` | About, Projects, Chat | Languages, frameworks, tools, domains of expertise |
| `education.md` | About, Chat | Degrees, certifications, courses, awards |
| `contact.md` | Contact, Chat | How people should reach you + which links to show publicly |

## Notes

- Anything you put here **will be used to generate publicly visible website content**,
  so don't include private info (home address, personal phone, secrets) unless you want
  it public. For the Ask AI feature, this content may also be sent to an LLM.
- Leave a file empty and I'll simply skip that section.
